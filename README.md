# oh-my-dotfiles
用来存放自己各种配置文件的仓库

---

## 食用指南
将仓库 clone 下来之后，在项目文件夹下使用 `install.sh` 安装即可：

```bash
./install.sh
```

> ⚠️ 安装前请务必对自己原本的配置文件备份！
